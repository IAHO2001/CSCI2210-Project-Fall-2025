Compile and Execute Instructions For GreyhoundEATS.

Step 1: Download the GreyhoundEats.zip file from GitHub under the Stage 4 folder of the main GitHub repository named CSCI2210-Project-Fall-2025.

Step 2: After downloading the FoodDeliverySystemCodesUpdates.zip from the Stage 4 folder of the main GitHub (CSCI2210-Project-Fall-2025), then place the folder somewhere easy to access. For instance I put this file on my desktop.

Step 3: Open command terminal on your operating system, then find the filepath for your folder with the codes for the food delivery system. For instance since mine was under desktop this is how my filepath for this folder looks like: 
  C:\Users\15056\OneDrive\Desktop\GreyhoundEATS.

Step 4: After you find your respective file path type the following: cd filepath (if this is confusing look at the example). 
  Ex: 

Example: cd C:\Users\15056\OneDrive\Desktop\GreyhoundEats.

Step 5: After you are placed into this directory/filepath then run the following command: java -jar "GreyhoundEats.jar"

Step 7: you are now in teh system .
