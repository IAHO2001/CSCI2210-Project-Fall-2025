Compile and Execute Instructions For The Food Delivery System code.

Step 1: Download the CS2210_FoodDeliverySystem.zip file from GitHub under the Stage 3 folder of the main GitHub repository named CSCI2210-Project-Fall-2025.

Step 2: After downloading the CS2210_FoodDeliverySystem.zip from the Stage 3 folder of the main GitHub (CSCI2210-Project-Fall-2025), then place it somewhere easy to access. For instance I put this file on my desktop.

Step 3: Open command terminal on your operating system, then find the filepath for your folder with the codes for the food delivery system. For instance since mine was under desktop this is how my filepath for this folder looks like: 
  C:\Users\15056\OneDrive\Desktop\CS2210_FoodDeliverySystem.

Step 4: After you find your respective file path type the following: cd filepath (if this is confusing look at the example). 

Example: cd C:\Users\15056\OneDrive\Desktop\CS2210_FoodDeliverySystem

Step 5: After you are placed into this directory/filepath then run the following command: javac *.java (This will compile all java files in the folder). 

Step 6: Next, Run the following command: java Main (This will execute the files)


Step 7: Lastly, when prompted chose your option from the Food Delivery System List.
